Beacon for iOS

Installation

Beacon is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod 'Beacon'

Initialize

Initialize the Beacon SDK with your Beacon ID

HSBeaconSettings *settings = [[HSBeaconSettings alloc] initWithBeaconId:@"beacon-id"];
[HSBeacon openBeacon:settings];

Launch Beacon

[HSBeacon openBeacon:settings];

Authenticating Users

Authenticating with Beacon is optional, but if you have information you’d like to associate with users, you can do so using HSBeaconUse and the login: method before showing the Beacon:

HSBeaconUser *user = [[HSBeaconUser alloc] init];
user.email = self.emailTextField.text;
user.name = self.nameTextField.text;

[HSBeacon login:user];

HSBeaconSettings *settings = [[HSBeaconSettings alloc] initWithBeaconId:@"beacon-id"];
[HSBeacon openBeacon:settings];

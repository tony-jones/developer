New Mailboxes - Beta

Your Help Scout mailbox experience is about to be much faster! We’ve spent the last few months refactoring mailboxes from the ground up, making all conversation actions and general mailbox updates much snappier across the board.

More than 10,000 lines of code were refactored as part of this project, which is why we need your help before releasing mailboxes to all Help Scout customers. We’d love for you to start using the new version of mailboxes today!

Accessing the Beta

Our team has been using the beta since mid-October. We don’t expect any problems or issues related to sending messages or updating conversations, as none of that has changed. It’s safe! If something breaks, it will likely be related to the user interface.

Anyone on your team can help test the new mailboxes. Here’s how:

  1. Visit https://secure-beta.helpscout.net/
  2. Log in with your standard Help Scout credentials
  3. Get to work!

All of your account data is the same between the beta environment and the current production environment. Any changes you make while using the beta URL will be available if you switch back to the standard URL.

Outstanding Issues

None that we’re aware of right now!

Safety Net

If the app feels sluggish for any reason or if you run into any errors, please let us know. Keep in mind you can continue where you left off by reverting back to the standard Help Scout URL: https://secure.helpscout.net. Your work between the two sites will always be in sync.

Next Up

This is the first big milestone in a series of improvements that are coming to Help Scout. Once the new mailboxes are available for everyone and feeling stable, we’ll start pushing real-time activity (folder counts, new messages, updates) to mailboxes as it happens.

Feedback

If you’ve discovered a bug or would like to share some general feedback, send an email to help@helpscout.net or click the Get Help link in the app to get in touch. Your feedback is super important, so don’t be shy. We’d love to hear from you!
